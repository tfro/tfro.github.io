On-line teaching and learning

·         Example SCORM Module

This is an example of a SCORM Module (Sharable Content Object Reference Model).  It is a narrated PowerPoint file.  Narration can be added just with the features built into Powerpoint, but I have a set of custom macros that I’ve created to make adding and editing the audio much easier.  The results are then packaged into the SCORM module using a free Powerpoint Adding called iSpring Free.  The resulting SCORM Module can then be uploaded to lerarning management systems such as Moodle (Coursespces at UVic).  The results can also be produced to a video.
